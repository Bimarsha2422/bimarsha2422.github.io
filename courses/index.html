courses

A running list of university courses I have completed, along with key topics covered in selected courses.

Computer Science (CS)

Introduction to Computer Science (CS-UH 1001)

Python programming fundamentals, Linux command line, data types, control structures (decision structures, loops), data structures (lists, dictionaries, sets), functions, modules, file handling, plotting, Object-Oriented Programming (OOP), recursion, testing, debugging.

Discrete Mathematics (CS-UH 1002)

Propositional logic and logical equivalence, set theory, counting techniques, proof techniques (induction, contradiction, case analysis), number theory, modular arithmetic, recurrence relations, functions, Big-O notation, discrete probability, graph theory.

Data Structures (CS-UH 1050)

C++ programming (pointers, classes, inheritance), algorithm analysis, asymptotic notation, linear structures (linked lists, stacks, queues), non-linear structures (binary search trees, AVL trees, heaps), hashing, maps and dictionaries, sorting algorithms, graph representations.

Algorithms (CS-UH 1052)

Asymptotic algorithm analysis, graph algorithms (traversals, shortest paths like Dijkstra's), greedy algorithms, minimum spanning trees, divide and conquer (mergesort), dynamic programming (knapsack problem), network flow, computational complexity (NP-completeness).

Computer Systems Organization (CS-UH 2010)

C programming, data representation (integer, floating-point), machine-level code and assembly language, program optimization, memory hierarchy and caching, virtual memory, linking.

Mathematics (MATH)

Multivariable Calculus with Applications to Science and Engineering (MATH-UH 1020)

3D vector geometry, partial differentiation, gradients and directional derivatives, optimization, Lagrange multipliers, multiple integration (double, triple), vector calculus, line and surface integrals, the theorems of Green, Stokes, and Gauss.

Foundations of Mathematics (MATH-UH 1010)

Formal logic, proof techniques, mathematical induction, set theory, partitions and equivalence relations, functions and relations, cardinality and infinite sets, axiomatic construction of number systems.

Linear Algebra (MATH-UH 1022Q)

Systems of linear equations, Gaussian elimination, matrix algebra, determinants, vector spaces, dimension and rank, linear transformations, eigenvalues and eigenvectors, diagonalization, orthogonality, Gram-Schmidt process.

Ordinary Differential Equations (MATH-UH 2010)

First-order differential equations, second-order linear equations, non-homogeneous equations (undetermined coefficients, variation of parameters), series solutions, Laplace transforms, systems of first-order linear equations, qualitative analysis (phase plane).

Probability and Statistics (MATH-UH 2011Q)

Descriptive statistics, probability theory, discrete and continuous random variables, Central Limit Theorem, parameter estimation and confidence intervals, hypothesis testing, regression and correlation, Bayesian methods.
